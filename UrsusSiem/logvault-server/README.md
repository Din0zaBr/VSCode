# URSUS SIEM

**URSUS SIEM** — система управления информационной безопасностью и событиями (SIEM) с открытым исходным кодом, разработанная на стеке FastAPI + PostgreSQL + React. Обеспечивает централизованный сбор, хранение, корреляцию и анализ журналов событий от множества агентов.

---

## Возможности

| Категория | Описание |
|-----------|----------|
| **Сбор событий** | REST API для приёма пачек логов, Syslog UDP/TCP, CEF |
| **Хранение** | PostgreSQL с JSONB-метаданными и полнотекстовым поиском |
| **Анализ** | Язык запросов PDQL, группировка, агрегация |
| **Корреляция** | Пороговые правила с поддержкой SIGMA-формата |
| **Оповещения** | Webhook, Telegram, пороговые алерты |
| **Агенты** | Лёгкие Python-агенты для Linux, управление API-ключами |
| **RBAC** | Роли admin / operator, изоляция по агентам |
| **Интеграции** | Active Directory, Kaspersky EDR, PT Sandbox, PT NAD |
| **UI** | React 18 + TanStack Query, 17 страниц дашборда |

## Быстрый старт

```bash
# Клонирование и запуск
git clone <url>
cd logvault-server
cp .env.example .env      # отредактируйте переменные под своё окружение
bash start.sh             # или: docker compose up --build -d
```

После запуска:
- **UI**: http://localhost:8080
- **API**: http://localhost:8000/api
- **Документация API** (Swagger): http://localhost:8000/docs

Учётные данные по умолчанию: `admin` / `admin` (смените при первом входе).

## Структура проекта

```
logvault-server/
├── server/          # Backend: FastAPI + Python
│   ├── src/
│   │   ├── routers/     # 15 групп API-эндпоинтов

│   │   └── integrations/ # Коннекторы (AD, EDR, Syslog, CEF)
│   └── init.sql     # Схема базы данных
├── ui/              # Frontend: React 18 + Vite + TypeScript
│   └── src/pages/   # 17 страниц дашборда
├── Docs/            # Документация (этот раздел)
├── docker-compose.yml
├── .env.example
└── start.sh
```
